Welcome to Huia.

Huia, is a programming language targetting the [Rubinius](http://rubini.us) VM.

## Installation

Add this line to your application's Gemfile:

    gem 'huia'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install huia
